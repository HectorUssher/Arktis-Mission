In meinem Programm geht es um eine Expedition zur Arktis um einen Stützpunkt zu überprüfen. Die genaue Hintergrundgeschichte wird während des Spiels erzählt. Man muss in mehreren Zügen an ein Ziel kommen, dass man aber während des Spiels erst noch herausfinden muss, indem man Hinweise findet. Während des Spiels bewegt man sich meistens in einem Schiff auf dem Wasser, kann aber auch auf Eisschollen und auf dem Festland an Land gehen. Dabei trifft man Gegner und kann Truhen mit Ressourcen finden. Die genauen Regeln werden während des Spiels erklärt. Das Spiel endet wenn entweder das Ziel erreicht wird oder wenn die Mission nicht mehr erfüllt werden kann, z.B. wenn das Schiff sinkt oder alle Crewmitglieder gestorben sind.
